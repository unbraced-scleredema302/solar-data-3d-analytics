## 🧭 Overview

solar-data-3d-analytics is a Windows app for viewing solar audit data in 3D. It turns energy data into a surface mesh view so you can spot voltage changes, battery health issues, and weak points in a solar setup.

Use it when you want a clear visual view of energy data instead of a long table of numbers. The app is built for fast viewing and simple use.

The app is built to keep the process simple.

The display shows the data as a 3D surface. Higher and lower points help you see where readings change.

### 3. Check voltage patterns
Use the visual shape to find voltage spikes, drops, and uneven areas.

### 4. Inspect battery health
Look for signs of battery wear or poor charge patterns in the mesh.

### 5. Compare sections
Move across the model to compare different parts of the solar system or audit area.
